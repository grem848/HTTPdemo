13. Explain (conceptually) about the TCP/IP-protocol stack
Works as the transportation layer of a connection, divides data into packages. Handles host-to-host

14. Explain about the HTTP Protocol (the following exercises will go much deeper into this protocol)
HTTP is a web protocol, with a set of rules, used to fetch resources like html. Part of the application layer

15. Explain (conceptually) how HTTP and TCP/IP are connected (what can HTTP do, and where does it fit into TCP/IP)
HTTP is the actual content, where the lower layers do transportation and communication.
Data coded according to application layer protocols are encapsulated into a transport layer protocol (such as TCP or UDP),
which in turn use lower layer protocols to effect actual data transfer.
